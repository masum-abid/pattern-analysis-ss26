# Pattern Analysis — Practical Exercises

A collection of hands-on coding assignments and mini-projects for the Pattern Analysis lecture series.
Each lecture folder contains self-contained Jupyter notebooks with code, visualizations, and explanations.

## Quick Start

```bash
git clone https://github.com/masum_abid/pattern-analysis-ss26.git
cd pattern-analysis-ss26
pip install -r requirements.txt
jupyter notebook
```
